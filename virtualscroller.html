VirtualScroller v1.0.0

High-performance infinite scrolling module for Roblox User Interfaces.

Get on Creator Store View Source Roblox DevForum

Introduction

VirtualScroller employs virtualization—only rendering the items currently visible to the user—to handle heavy lists without dropping frames.

Installation

Download the latest model file from the repository or insert via Roblox Library. Require the module in your script.

Usage Example

local VirtualScroller = require(path.to.VirtualScroller) -- 1. Create a scroller instance local Scroller = VirtualScroller.new({ Frame = playerGui.Frame, Template = script.ItemTemplate, Padding = 5 }) -- 2. Generate large dataset local data = {} for i = 1, 10000 do table.insert(data, { Name = "Item " .. i, Rarity = "Common" }) end -- 3. Load data instantly Scroller:SetData(data)

API Reference

VirtualScroller.new(config: Table) -> Scroller
Creates a new scroller attached to the target ScrollingFrame.
Scroller:SetData(dataArray: Table)
Clears current list and populates the scroller with new data. Automatically calculates scroll height.
Scroller:AddItem(itemData: Table)
Appends a single item to the end of the list without refreshing the entire view.
Scroller:RemoveItem(index: number)
Removes the item at the specific index and shifts subsequent items up.
Scroller:Clear()
Empties the data array and resets the canvas size to 0.
Scroller:Destroy()
Disconnects all render events and cleans up the metatable.
